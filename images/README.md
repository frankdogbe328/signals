# Images Folder

## Logo File

Place your logo file here with the name: **`logo.png`**

### Supported Formats:
- PNG (recommended)
- JPG/JPEG
- SVG

### File Name:
The logo should be named exactly: **`logo.png`**

If your logo is in a different format, you can:
1. Rename it to `logo.png` (if it's already PNG)
2. Convert it to PNG format
3. Or update the HTML files to use your logo's actual filename

### Logo Sizes:
- **Login page**: Maximum 200px width, 120px height (auto-scales)
- **Dashboard navbar**: Maximum 50px height (auto-scales)

### How to Add Your Logo:
1. Copy your logo file
2. Paste it into this `images` folder
3. Rename it to `logo.png` (or update HTML if using different name)
4. The logo will automatically appear on all pages!
