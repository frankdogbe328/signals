# Exam Portal

This is the exam/quiz system for the Signal Training School LMS.

## Structure

- `lecturer-exam-dashboard.html` - Lecturer portal for creating and managing exams
- `student-exam-portal.html` - Student portal for taking exams
- `css/` - Stylesheets (will link to main CSS or have separate styles)
- `js/` - JavaScript files for exam functionality
- `images/` - Exam-related images

## Features

- Multiple question types (Multiple Choice, True/False, Short Answer, Essay)
- Time-limited exams with countdown timer
- One-way navigation (no going back)
- Randomized questions per student
- Auto-grading
- Manual result release by lecturers
- Linked to LMS database (shared students and subjects)
