üìã Document Format Guide

Format your questions as follows:

1. What is the capital of Ghana? A. Kumasi B. Accra C. Tamale D. Takoradi Answer: B 2. True or False: Ghana is in West Africa. Answer: True 3. Explain the importance of communication in the military. Answer: [Your expected answer here]

Tips: Use numbered questions (1, 2, 3...), mark options with A, B, C, D, and include "Answer:" followed by the correct answer. For multiple choice, options should be on separate lines.

Only .docx files are supported (not .doc). You can save older .doc files as .docx in Word.

üìã Excel Format Guide

Format your Excel file with these columns:

Question Type Option A Option B Option C Option D Correct Answer Marks
What is the capital of Ghana? multiple_choice Kumasi Accra Tamale Takoradi B 1
Ghana is in West Africa. true_false - - - - True 1

Types: multiple_choice, true_false, short_answer, essay
Note: For multiple_choice, fill Option A-D. For other types, leave options blank. Correct Answer should be the letter (A, B, C, D) for multiple choice, or "True"/"False" for true/false questions.

Only .xlsx and .xls files are supported. First row can be headers (will be skipped automatically).
